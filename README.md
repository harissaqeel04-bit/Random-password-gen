# Password Generator

A clean and simple password generator made using **HTML, CSS, and JavaScript**.  
Users can generate secure passwords by selecting length and character options.

##  Features
- Adjustable password length  
- Include uppercase, lowercase, numbers, symbols  
- Password strength indicator  
- Copy-to-clipboard  
- Modern responsive UI  

##  Files
- `index.html` – App structure  
- `styles.css` – UI styling  
- `script.js` – Password generation logic  

##  Usage
Open **index.html** in any browser and start generating passwords.
